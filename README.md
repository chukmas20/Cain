Organization Homepage
A clean, responsive organization homepage built with plain HTML and CSS — no frameworks or dependencies required.
Project structure
├── index.html      # Main HTML file (structure & content)
├── styles.css      # External stylesheet (layout, design, responsiveness)
└── README.md       # Project documentation

Features
Header — Sticky top bar with logo and navigation links
Navigation Menu — Desktop nav with a highlighted CTA button; collapses into an animated hamburger menu on mobile
Hero Section — Bold headline, description, call-to-action buttons, and a stats panel
About Section — Organization mission, story, and a 4-value card grid
Footer — Multi-column layout with quick links, services, and contact info
Responsive Design — Fully adapts to desktop, tablet, and mobile using CSS media queries
